=====[DATA STRUCTURES]===========================================================

- DATA STRUCTURES ARE WAYS OF STORING 
ORGANIZED DATA SO THAT IT CAN BE USED 
EFFICIENTLY.

- PRIMITIVE DATA STRUCTURES STORE A SINGLE 
VALUE AND ARE ALSO REFERRED TO AS SIMPLY 
DATA TYPES. tHESE INCLUDE CHAR, INT, 
FLOAT ETC. 

- NON-PRIMITIVE DS => LINEAR VS NON-LINEAR

- NON-LINEAR => TREES AND GRAPHS

- LINEAR => STATIC VS DYNAMIC

- STATIC => ARRAYS, HASHES

- DYNAMIC => LINKED LISTS, STACKS, QUEUES
