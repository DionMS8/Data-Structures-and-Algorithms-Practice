=====[TREES]===========================================

- A TREE IS A HIERARCHIAL DATA STRUCTURE WHICH 
CONTAINS A COLLECTION OF NODES. EACH NODE IS 
A DATA STRUCTURE WHICH CONTAINS A VALUE AND A 
LIST OF REFERENCE NODES/CHILDREN.

- THE TREE STARTS AT THE ROOT NODE 
- NO REFERENCE NODE IS DUPLICATED.
- NO REFERENCE NODE POINTS TO THE ROOT NODE.
