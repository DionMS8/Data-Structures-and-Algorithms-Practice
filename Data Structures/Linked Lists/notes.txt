=====[LINKED LISTS]===========================================

- A LINKED LIST IS A DATA STRUCTURE THAT 
CONTAINS A SEQUENCE OF NODES. 
- EACH NODE CONTAINS A DATA FIELD AND A 
REFERENCE POINTER.
- EACH NODE IS CONNECTED TO THE NEXT NODE 
USING THE REFERENCE POINTER, WHICH FORMS A 
CHAIN-LIKE LINK FOR DATA STORAGE.
